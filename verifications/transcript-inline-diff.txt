# Verification — inline transcript diffs (capped, syntax-highlighted, ctrl+o)

Feature: a successful file-mutating tool result (write_file/edit_file/
delete_file) now renders its diff inline in the transcript, GitHub-PR style
via the shared `crate::diff` renderer — header rule with the path, gutter
line numbers, add/remove backgrounds with syntax colors layered under, and
a closing rule with the +added/-removed counts. Collapsed shows at most
INLINE_DIFF_CAP (10) styled diff lines with a `⋯ +N more diff lines ·
ctrl+o opens the full diff` fold hint; ctrl+o (the existing per-entry
expansion in the deck's Session tab, ctrl+r expand-all in the classic
shell) reveals the whole diff. Freshness-gated: a later mutation of the
same path bumps `FileState::changes` past the recorded `InlineDiffRef.seq`
and the diff is hidden rather than misattributed.
